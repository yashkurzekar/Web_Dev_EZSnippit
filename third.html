Web Page

A web page is web document thay is accessed in a web browser. A website typically contains many web pages linked together under a common domain name. Tjhe term "Web page" is therefore a metaphot of paper pages bond together into a book.

Navigation

Each web page is identified by a distinct Unifirm Resourcce Locator(URL). When the user imports a URL into their browser, the browser retrieves the necessary contect from a web server and then transform it into an interactive visal representation on the user's screen.

If the user clicks or taps a link, the browser repeats this process to load tge new URL, which could be part of the current website or a different one. The browser has features, such as the address bar, that indicate which page is displayed.
